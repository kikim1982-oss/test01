# 매콤 알리오올리오 (Spicy Aglio e Olio)

> ⏱️ 조리시간: 15분 | 🍽️ 1인분 | 난이도: ⭐ 쉬움

## 📝 재료
| | |
|------|------|
| 스파게티 면 — 100g | 마늘 — 4~5쪽 |
| 올리브유 — 3~4큰술 | 소금 — 약간 (면 삶는 물용) |
| 후추 — 약간 | 페페론치노(건고추) — 2~3개 *(매운맛의 핵심!)* |
| 고추기름 — 1큰술 *(없으면 고춧가루 1작은술로 대체)* | 청양고추 — 1개 *(송송 썰기, 없으면 생략 가능)* |
| 파슬리 — 약간 *(없으면 생략 가능)* | |

## 👨‍🍳 만드는 법

![cooking](./thumbnails/spicy-aglio-e-olio-cooking.png)

| 🍽️ 맛있게 만드는 법 | 🥗 저칼로리로 만드는 법 |
|------|------|
| 1. **물 끓이기** — 냄비에 물을 넉넉히 붓고 강불로 끓입니다. 소금 한 큰술 투입! | 1. **물 끓이기** — 동일하게 진행. 소금은 1/2큰술로 줄여주세요. |
| 2. **마늘 준비** — 마늘을 얇게 슬라이스. 칼 옆면으로 으깬 뒤 썰면 더 쉬워요. | 2. **마늘 준비** — 동일. 마늘은 칼로리 거의 없으니 넉넉히! |
| 3. **면 삶기** — 봉지 표기 시간보다 1분 일찍 건져내세요. 면수 한 국자 남겨두기! | 3. **면 삶기** — 스파게티 대신 곤약면이나 통밀면(80g)으로 대체하면 칼로리 대폭 감소! |
| 4. **마늘 & 고추 볶기** — 올리브유+고추기름을 두르고 약불로 마늘, 페페론치노, 청양고추를 볶기. 매운 향이 올라오면 성공! | 4. **마늘 & 고추 볶기** — 올리브유 1작은술만 사용. 고추기름 제거. 논스틱 팬 활용! |
| 5. **면과 소스 합치기** — 삶은 면을 프라이팬에 옮기고 면수를 조금씩 넣으며 섞기. | 5. **면과 소스 합치기** — 동일. 면수를 넉넉히 넣어 기름 부족을 보완! |
| 6. **마무리** — 후추, 파슬리 올려 완성! | 6. **마무리** — 후추, 파슬리 올려 완성! |

## 💡 꿀팁

| 🍽️ 맛있게 꿀팁 | 🥗 저칼로리 꿀팁 |
|------|------|
| **면수는 필수!** 면수 한 국자(약 100ml)를 꼭 남겨두세요. | 곤약면 사용 시 면수가 없으니 물 100ml에 소금 약간 섞어 대체! |
| **마늘은 약불로 천천히** — 강불에서 볶으면 금방 타버려요. | 올리브유 1작은술 + 논스틱 팬이면 기름 걱정 끝! |
| **설거지 최소화** — 냄비 하나로 완성하면 설거지가 확 줄어요! | 설거지 최소화는 동일! 냄비 하나 전략 추천. |
| **매운맛 조절** — 페페론치노를 부숴서 씨까지 넣으면 더 매워요! | 고추기름 대신 고춧가루만 써도 매운맛은 유지하면서 칼로리 절감! |
| **재료 대체** — 올리브유 없으면 식용유로 OK. | 통밀면이나 곤약면으로 바꾸면 칼로리 50% 이상 절감! |
| **면 종류** — 소면이나 다른 파스타 면으로도 OK! | 애호박면(줄리엔느)으로 대체하면 초저칼로리 완성! |
