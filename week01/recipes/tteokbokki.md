# 떡볶이

> 조리시간: 15분 | 1인분 | 난이도: 쉬움

## 📝 재료
| | |
|------|------|
| 떡볶이 떡 — 200g | 어묵 — 1장 (없으면 생략 가능) |
| 물 — 1컵 (200ml) | 고추장 — 1.5 큰술 |
| 간장 — 1 작은술 | 설탕 — 1 큰술 |
| 대파 — 약간 (없으면 생략 가능) | |

## 👨‍🍳 만드는 법

![cooking](./thumbnails/tteokbokki-cooking.png)

| 🍽️ 맛있게 만드는 법 | 🥗 저칼로리로 만드는 법 |
|------|------|
| 1. 떡이 딱딱하다면 찬물에 5분간 담가 부드럽게 만들기. | 1. 떡볶이 떡 대신 곤약떡(150g)을 사용! 물에 헹궈 비린내 제거. |
| 2. 어묵은 먹기 좋은 크기로 잘라 주세요. | 2. 어묵 대신 두부나 삶은 달걀로 단백질 보충! |
| 3. 냄비에 물 1컵, 고추장, 간장, 설탕을 넣고 잘 섞기. | 3. 설탕 → 알룰로스로 대체. 고추장도 1큰술로 줄이기. |
| 4. 중불로 소스가 끓으면 떡과 어묵 투입! | 4. 동일하게 진행. 양배추, 팽이버섯 등 채소를 추가하면 포만감 UP! |
| 5. 중약불로 7~8분간 저어가며 졸이기. | 5. 동일. 곤약떡은 빨리 익으니 5분이면 충분! |
| 6. 소스 걸쭉, 떡 말랑하면 완성! 대파 올리기. | 6. 완성! 대파 올려서 마무리. |

## 💡 꿀팁

| 🍽️ 맛있게 꿀팁 | 🥗 저칼로리 꿀팁 |
|------|------|
| 냄비 하나만 쓰면 설거지 최소화! | 곤약떡은 칼로리가 거의 0에 가까워요! |
| 떡이 눌어붙지 않게 중간중간 저어주세요. | 채소(양배추, 팽이버섯)를 넣으면 양은 늘고 칼로리는 줄어요. |
| 고추장 양으로 매운맛 조절 가능! | 알룰로스가 없으면 스테비아나 설탕 1/2큰술로 대체! |
| 어묵 대신 삶은 달걀, 햄, 두부도 맛있어요. | 남은 소스에 곤약밥을 비벼 먹으면 저칼로리 마무리 완성! |
| 남은 소스에 밥을 비벼 먹으면 꿀맛! | |
