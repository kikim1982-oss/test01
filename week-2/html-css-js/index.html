CSS 14. Overflow (넘침 처리)

visible
넘쳐도 그대로 보입니다. 텍스트가 박스를 넘어갑니다.
hidden
넘치는 부분이 잘려서 안 보입니다. 숨겨진 텍스트.
scroll
항상 스크롤바가 표시됩니다. 내용이 넘치지 않아도요.
auto
넘칠 때만 스크롤바가 나타납니다. 가장 많이 사용합니다.

CSS 15. CSS 변수 (Custom Properties)

:root에서 --변수명: 값으로 선언하고, var(--변수명)으로 사용합니다.

background: var(--primary) = #667eea
background: var(--secondary) + box-shadow: var(--shadow)
border-color: var(--danger) + color: var(--danger)

변수를 쓰면 색상이나 값을 한 곳에서 관리할 수 있어 유지보수가 편합니다!

7. 시맨틱 태그

시맨틱 태그는 페이지 구조에 의미를 부여합니다:

<header> - 머리말 영역 (로고, 네비게이션 등)
