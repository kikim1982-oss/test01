# 간단 볶음밥

> *"냉장고의 남은 재료가 하나의 접시 위에서 완성을 만날 때, 비로소 요리가 시작된다."*

볶음밥은 동아시아 가정 요리의 정수다. 전날 밤의 찬밥, 서랍 속에 잠든 채소 한 줌, 계란 하나로 빚어내는 이 한 접시는 미쉐린 셰프의 주방에서도 여전히 존경받는 기술을 요한다. 강불에서 빠르게 완성되는 볶음밥은 불 조절과 타이밍이 모든 것을 결정하며, 간결한 재료 구성이야말로 본연의 맛을 살리는 정석이다.

---

**조리 시간** 12분 · **서빙** 1인분 · **난이도** Easy

---

## Ingredients

| | |
|:------|:------|
| 밥 (찬밥 권장) — 1공기 (200g) | 계란 — 2개 |
| 대파 — 1/3대 (약 30g) | 당근 — 1/4개 (약 40g) |
| 냉동 완두콩 또는 옥수수 — 2큰술 | 식용유 — 2큰술 |
| 간장 — 1큰술 | 참기름 — 1/2작은술 |
| 소금 — 약간 | 후추 — 약간 |

---

## Method

![cooking](./thumbnails/fried-rice-cooking.png)

| Classic | Light |
|:------|:------|
| **01** 대파는 송송 썰고, 당근은 5mm 크기로 잘게 다진다 | **01** 대파는 송송 썰고, 당근은 5mm 크기로 잘게 다진다 |
| **02** 팬을 강불로 가열 후 식용유 2큰술을 두르고, 당근과 완두콩을 30초간 볶는다 — 수분을 날리는 것이 관건이다 | **02** 팬에 식용유 1작은술만 두르고, 당근과 완두콩을 30초간 볶는다 |
| **03** 계란 2개를 깨뜨려 넣고 스크램블하되, 완전히 익히기 전 반숙 상태에서 밥을 투입한다 — 이 타이밍이 촉촉한 식감의 비결이다 | **03** 계란 1개만 사용하고, 완전히 익힌 후 밥을 투입한다 |
| **04** 밥을 넣고 주걱으로 눌러가며 팬 전체에 고르게 펼쳐 1분간 강불에 볶는다 — 밥알이 서로 분리될 때까지 볶을 것 | **04** 밥을 넣고 중불에서 1분 30초간 꼼꼼히 볶아 수분을 날린다 |
| **05** 간장 1큰술을 팬 가장자리로 흘려넣어 직화로 향을 입힌 후, 대파를 넣고 10초간 마무리 볶음 | **05** 저염 간장 1/2큰술로 간하고, 대파를 넣어 마무리한다 |
| **06** 불을 끄고 참기름 1/2작은술, 소금·후추로 최종 간을 맞춘 후 즉시 서빙한다 | **06** 불을 끄고 참기름 1/4작은술만 사용, 소금·후추로 마무리한다 |

## Chef's Note

| Classic Tips | Light Tips |
|:------|:------|
| 찬밥을 사용할 것 — 갓 지은 밥은 수분이 많아 볶음밥이 뭉침. 전날 밥을 냉장 보관했다가 사용하면 최적 | 밥을 반 공기(100g)로 줄이고 채소와 달걀 비중을 높이면 포만감은 유지하면서 칼로리 절감 가능 |
| 간장은 팬 가장자리 직화로 투입할 것 — 간장의 아미노산이 고온에서 마이야르 반응을 일으켜 풍미가 배가됨 | 간장 대신 참치액(저염)이나 굴소스 소량으로 감칠맛을 살릴 것 |
| 강불 유지가 핵심 — 중불 이하로 낮추면 볶음밥이 아닌 찜밥이 됨. 연기가 살짝 날 정도의 고온 유지 | 식용유 사용량을 최소화하려면 코팅 팬 사용 권장 — 기름 1작은술로도 달라붙지 않음 |
| 동선 팁: 재료는 사전에 모두 손질 후 팬 옆에 대기. 볶음은 멈추지 않고 연속으로 진행할 것 | 계란 흰자만 사용하면 콜레스테롤을 줄이면서도 단백질은 그대로 유지 가능 |
