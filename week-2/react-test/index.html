텍스트 관련 태그

제목 태그 (h1 ~ h6)

이것은 strong (굵은 글씨) 태그입니다.

이것은 em (기울임) 태그입니다.

이것은 mark (형광펜) 태그입니다.

이것은 del (취소선) 태그입니다.

이것은 small (작은 글씨) 태그입니다.

이것은 a (링크) 태그입니다.

H2O 는 sub 태그, E=mc2 는 sup 태그입니다.

blockquote 태그는 인용문을 표시할 때 사용합니다.

위의 <hr> 태그는 수평선을 만듭니다.

리스트 태그

순서 없는 리스트 (ul)

  • HTML - 구조
  • CSS - 스타일
  • JavaScript - 동작

순서 있는 리스트 (ol)

  1. 먼저 HTML을 배운다
  2. 다음으로 CSS를 배운다
  3. 마지막으로 JavaScript를 배운다

설명 리스트 (dl)

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
